# Changelog

### Initial Release

- 📋 **TODO Aggregator** — Scan for TODO/FIXME/HACK/XXX/BUG/NOTE, prioritized display

- 📝 **Quick Notes** — Per-project markdown scratchpad
- ⚡ **Dashboard** — Stats overview with quick action buttons
